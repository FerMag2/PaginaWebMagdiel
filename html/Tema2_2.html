Concepto de índice

Estructura especial diseñada para mejorar la velocidad de recuperación y búsqueda de datos en una tabla. Funciona de manera similar a un índice en un libro.

Procedimientos

  • Un conjunto de instrucciones o un programa almacenado en la base de datos que realiza una tarea específica o una serie de tareas relacionadas.
  • Se utilizan para automatizar tareas comunes, ejecutar consultas complejas, realizar operaciones de actualización en los datos.
